# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-14

### Added

- Initial release of mfk-mask-input-antd
- AntdInputMask component with full IMask integration
- TypeScript support with complete type definitions
- Support for all Ant Design Input props
- Multiple mask format support (phone, credit card, date, custom)
- Access to both masked and unmasked values in onChange
- Comprehensive documentation and examples

### Features

- Flexible masking with IMask
- Seamless Ant Design integration
- Peer dependencies for React, Ant Design, and IMask
- ESM and CommonJS builds
- Full TypeScript declarations
